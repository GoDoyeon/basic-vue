# 컴포넌트 데이터 컴포넌트의 data는 반드시 함수여야 합니다.

컴포넌트(new Vue를 제외한 모든 곳)의 data 프로퍼티의 값은 반드시 객체를 반환하는 함수여야 한다.

### Bad Code

```
Vue.component('some-comp', { 
  data: { 
    foo: 'bar' 
  }
 })
 ```

### Good Code

```
Vue.component('some-comp', { 
  data: function() { 
    return { 
      foo: 'bar' 
    }
  }
})
```

```
new Vue ({ 
  data: { 
    foo: 'bar' 
  } 
})
```
